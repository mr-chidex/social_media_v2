## Social Media API

A Node JS service RESTful API. Social Media ApI

## Built With

- Node.js
- Postgres
- TypeORM
- Express
- Git
- Json Web Token
- Postman
- TypeScript
- Cloudinary

## Clone this project

```
git clone https://github.com/mr-chidex/social_media_v2.git
```

```
cd social_media_v2
```

## Install dependencies

```
yarn install
```

### Running this project locally

```
yarn dev
```
